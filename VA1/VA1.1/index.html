HTML Head Tag

The Head Element

The head element is a container for metadata (data about data) and is placed between the html tag and the body tag.

The Title Element

The title element defines the title of a document, which is displayed in the browser's title bar or a page's tab.

The Link Element

The link tag defines the relationship between the current document and an external resource.

The Meta Element

The meta tag defines metadata about an HTML document. Metadata is data (information) about data. Meta tags always go inside the head.

Some of the meta tags are:

  • Specifies the character encoding for the HTML document

    • meta charset="UTF-8"

  • Provides an HTTP header for the information/value of the content attribute

    • meta http-equiv="refresh" content="30"

  • Define keywords for search engines

    • meta name="keywords" content="HTML, CSS, JavaScript"

  • Define a description of your web page

    • meta name="description" content="Free Web tutorials for HTML and CSS"

  • Setting the viewport to make your website look good on all devices

    • meta name="viewport" content="width=device-width, initial-scale=1.0"
